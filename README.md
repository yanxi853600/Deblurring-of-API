# Scale-recurrent Network for Deep Image Deblurring API

## Prerequisites
- Python3.6
- Scipy
- Scikit-image
- numpy
- Tensorflow 1.4 with NVIDIA GPU (RTX 2080 Ti super)

## Installation
Clone this project to your machine. 

```bash
git clone https://github.com/jiangsutx/SRN-Deblur.git
cd SRN-Deblur
```
